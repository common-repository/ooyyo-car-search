Search for car ads in OOYYO.com database
 
== Description ==
The OOYYO Car Search widget is designed for WordPress websites to allow their users to search for car ads in OOYYO.com database. 
Website owners can choose the default: 
<ul>
<li>country,</li>
<li>currency,</li>
<li>language</li>
</ul>
of the plugin, <br> while the users can select:
<ul>
<li>country,</li>
<li>car make,</li>
<li>model,</li>
<li>trim,</li>
<li>price range</li>
</ul>
<br>
The search results (car listings) are displayed in a full screen overlay. 
The interface is simple and responsive.
